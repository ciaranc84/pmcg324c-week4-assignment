Questions

1. What is the difference between `input type="text"` and `textarea`?
The input type="text" is different from textarea because the input type keeps it as a single line input. textarea is for multiline input.

2. What are two kinds of input you could use to have a user pick an option from multiple choices?
The radio input type and the select tag.

3. What are the two ways to add validation rules to a form field?
With built in form validation in HTML or you can use javascript to program custom validation.

4. When is data erased from `window.localStorage`?
When the history is cleared on your browser.

5. What's the difference between `JSON.stringify()` and `JSON.parse()`?
JSON.stringify takes Data and converts it from an object into a JSON string while JSON.parse brings it back to what it was.
